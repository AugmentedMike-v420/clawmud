# CM-012: Skills System

**Type:** feature  
**Priority:** high  
**Status:** Design Complete  

## Summary
Design and implement a cyberpunk-appropriate skills system. Skills represent learned abilities, improved through use and training. No fantasy tropes — this is guns, hacking, and street smarts.

## Requirements

### Skill Categories (by linked stat)

**BODY Skills:**
- [ ] Athletics — Running, jumping, climbing
- [ ] Endurance — Stamina, fatigue resistance
- [ ] Melee Combat — Hand-to-hand, blades, blunt
- [ ] Strength Feat — Lifting, breaking, forcing

**REFLEXES Skills:**
- [ ] Firearms — All ranged weapons
- [ ] Evasion — Dodging, avoiding
- [ ] Driving — Ground vehicles
- [ ] Stealth — Sneaking, hiding
- [ ] Pickpocket — Theft, sleight of hand

**TECH Skills:**
- [ ] Hacking — System intrusion
- [ ] Electronics — Device knowledge
- [ ] Repair — Fixing things
- [ ] Demolitions — Explosives
- [ ] Medicine — Medical care
- [ ] Pharmaceuticals — Drugs, poisons
- [ ] Cybertechnology — Cyberware work

**COOL Skills:**
- [ ] Persuasion — Convincing others
- [ ] Intimidation — Threatening
- [ ] Deception — Lying, disguise
- [ ] Streetwise — Street knowledge
- [ ] Perception — Noticing things
- [ ] Willpower — Mental resistance
- [ ] Performance — Entertainment, influence

### Skill Mechanics
- [ ] Skill check: 1d20 + Stat + Skill + Modifiers vs DC
- [ ] Skill levels: 0 (untrained) to 6 (legendary)
- [ ] Level bonuses: -2 at 0, +1 at 1, scaling to +10 at 6
- [ ] DC scale: 5 (trivial) to 30 (legendary)

### Advancement System
- [ ] Use-based: Successful challenging checks grant XP
- [ ] Training: Pay eddies for skill points
- [ ] Skillsofts: Cyberware chips provide instant skills

### Specializations
- [ ] Narrow bonuses within broad skills
- [ ] Example: Firearms → Pistols, Sniping
- [ ] Require base skill level 2-3

### Skill Synergies
- [ ] Related skills provide small bonuses
- [ ] Encourages breadth of knowledge

### Untrained Use
- [ ] Some skills can attempt untrained at -2
- [ ] Technical skills (hacking, medicine) require training
